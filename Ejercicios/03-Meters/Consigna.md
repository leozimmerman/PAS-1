# Consigna

Desarrollar una aplicación/plug-in que incluya un algoritmo de análisis y un medidor visual.

De ser posible crear un PullRequest con los archivos .h y .cpp en el directorio de este ejercicio dentro de una carpeta con el nombre del desarrollador.  

## Propuestas opcionales en base a las Apps de Meters analizadas: 

- Hacer mono canal. 
- Cambiar diseño de los medidores. 
- Cambiar algoritmo de análisis (descriptor)
- Agregar varios descriptores simultáneos, con varios medidores, o con un selector. 
- Transformar en plugin
